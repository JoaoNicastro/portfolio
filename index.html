Welcome

About Me

Hi, I'm João Nicastro Silveira — a computer science student with a passion for building things that work beautifully and thinking through problems from every angle. Whether I'm writing code, studying poker strategy, or diving into a new subject, I bring the same curiosity and persistence to everything I do.

What makes me unique is the way I blend analytical thinking with creative problem-solving. I discovered early on that programming is a lot like poker: both require patience, pattern recognition, and the ability to make good decisions with incomplete information. That mindset has shaped how I approach software development — I care about structure, long-term patterns, and building things that are both functional and elegant.

This semester has been especially transformative. Through my coursework in accessible web design, I've learned to write intentional, semantic HTML and to think deeply about how design choices affect real people. Features like skip links, proper heading hierarchies, and ARIA landmarks aren't just technical requirements — they're a way of respecting every user who interacts with your work. I'm proud of how much my UI/UX instincts have grown: cleaner layouts, smarter defaults, and less friction everywhere.

The purpose of this portfolio is to give you a window into who I am — as a student, a developer, and a person. Here you'll find my resume, highlights from my academic projects and coursework, a look at the interests and activities that shape my perspective, and links to my professional profiles online. I hope this site reflects the care and intentionality I bring to all my work. Thank you for visiting, and I invite you to explore and get to know me better.

Highlights of Academic Projects & Work

Key projects from my coursework that demonstrate my growth as a developer.

Universal Design Playground

A multi-page educational website built as my final project for ATLS/CSCI 3150. The site explores the principles of universal design and demonstrates fully accessible HTML and CSS — including semantic structure, ARIA landmarks, skip links, keyboard navigation, and responsive layouts. This project solidified my understanding of how design choices impact real users with diverse abilities.

View Live Site

HuckleSite — Full-Stack Web App

A React + TypeScript application built with Vite and deployed on Firebase. HuckleSite is a produce delivery concept that showcases component-based architecture, modern frontend tooling, and cloud deployment. Working on this project taught me how to manage state, build reusable components, and configure real hosting infrastructure beyond simple static pages.

View on GitHub

Accessible CSS Layout Lab

A hands-on lab project exploring CSS Grid and Flexbox to create multicolumn layouts with fully accessible navigation. This exercise focused on building responsive designs that maintain usability across screen sizes while preserving correct reading order and keyboard accessibility. It deepened my practical understanding of how layout techniques intersect with inclusive design.
